#  Ecomoneda

Proyecto web desarrollado como práctica de desarrollo frontend, enfocado en la creación de una plataforma visual moderna con diseño responsive y estructura limpia.

##  Vista previa

 Demo: https://facvsalazar.github.io/Ecomoneda/
 Repo: https://github.com/facvsalazar/Ecomoneda

##  Descripción

**Ecomoneda** es un proyecto web creado con fines educativos y de práctica, orientado al desarrollo de interfaces modernas utilizando tecnologías base del desarrollo web.

El objetivo principal fue aplicar conocimientos de:

* Estructura HTML semántica
* Estilos modernos con CSS
* Interactividad con JavaScript
* Organización de proyecto frontend

---

##  Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Diseño responsive

##  Lo que aprendí

Durante el desarrollo de este proyecto trabajé en:

* Maquetado web desde cero
* Organización de archivos frontend
* Diseño responsive básico
* Buenas prácticas en HTML y CSS
* Manipulación inicial del DOM con JavaScript
